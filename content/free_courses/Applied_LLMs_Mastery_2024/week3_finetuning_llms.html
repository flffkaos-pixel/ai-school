[3주차] LLM 미세 조정

ETMI5: 5에서 설명해주세요

이 섹션에서는 LLM에 대한 미세 조정 도메인 적응 방법을 살펴보겠습니다.

소개

미세 조정은 사전 훈련된 모델을 가져와 더 작은 도메인별 데이터 세트에서 추가로 훈련하는 프로세스입니다.

광범위한 NLP 작업을 위해 설계된 최첨단 LLM인 OpenAI의 GPT-3을 고려해 보세요.

이 시나리오에서 조직은 의료 보고서와 환자 기록으로 가득 찬 데이터세트에서 GPT-3를 미세 조정하는 작업에 참여합니다.

미세 조정은 언어 모델에만 국한되지 않습니다.

미세 조정의 핵심 원칙은 사전 훈련된 모델을 활용하고 새로운 데이터를 사용하여 매개변수를 재보정하여 새로운 상황이나 애플리케이션에 적응시키는 것입니다.

미세 조정을 수행하는 이유는 무엇입니까?

대규모 언어 모델은 실제로 다양한 작업 세트에 대해 학습되지만 이러한 대규모 일반 모델은 다양한 애플리케이션에서 합리적으로 잘 작동하도록 설계되었지만 특정 작업에서 반드시 뛰어난 것은 아니기 때문에 미세 조정의 필요성이 발생합니다.

모델이 특정 작업이나 관심 영역에서 뛰어난 숙련도를 달성하도록 하려면 미세 조정이 필수적입니다.

본질적으로 일반적인 대형 언어 모델은 여러 작업에 능숙하지만 어떤 작업에서도 숙달 수준에 도달하지 못하는 것으로 간주될 수 있습니다.

더 깊은 이해를 위해 몇 가지 설득력 있는 이유로 새로운 영역의 작업에 대한 모델 미세 조정이 중요하다고 간주되는 이유를 살펴보세요.

  1. 영역별 적응: 사전 훈련된 LLM은 특정 작업이나 영역에 최적화되지 않을 수 있습니다.
  2. 데이터 분포의 변화: 하나의 데이터 세트로 훈련된 모델은 분포를 벗어난 사례에 대해 잘 일반화되지 않을 수 있습니다.
  3. 비용 및 리소스 효율성: 새로운 작업에 대해 처음부터 모델을 학습하려면 라벨이 지정된 대규모 데이터 세트가 필요한 경우가 많으며 이는 비용과 시간이 많이 소요될 수 있습니다.
  4. 배포되지 않은 데이터 처리:
  5. 지식 이전:
  6. 작업별 최적화:
  7. 사용자 기본 설정에 따른 조정: 미세 조정을 통해 모델을 사용자 기본 설정 및 특정 작업 요구 사항에 맞게 조정할 수 있습니다.
  8. 지속적 학습: 미세 조정은 모델이 시간이 지남에 따라 진화하는 데이터와 사용자 요구 사항에 적응할 수 있도록 하여 지속적인 학습을 지원합니다.

요약하자면, 미세 조정은 조직이 사전 훈련된 모델을 특정 작업, 도메인 및 사용자 요구 사항에 맞게 조정하여 실제 애플리케이션에 모델을 배포하기 위한 실용적이고 효율적인 솔루션을 제공할 수 있는 강력한 기술입니다.

미세 조정 유형

높은 수준에서 언어 모델의 미세 조정 방법은 감독 및 비지도라는 두 가지 주요 접근 방식으로 분류될 수 있습니다.

비지도 미세 조정 방법:

  1. 비지도 완전 미세 조정: 비지도 미세 조정은 기존 동작을 수정하지 않고 LLM의 지식 기반을 업데이트해야 할 때 적합합니다.
  2. 대조 학습: 대조 학습은 언어 모델을 미세 조정하는 데 사용되는 방법으로, 잠재 공간에서 유사한 예와 다른 예를 식별하기 위한 모델 훈련을 강조합니다.

감독된 미세 조정 방법:

  1. 매개변수 효율적인 미세 조정: 언어 모델의 매개변수 업데이트와 관련된 계산 비용을 줄이는 것을 목표로 하는 미세 조정 전략입니다.
  2. 감독된 전체 미세 조정: 학습 과정 중에 언어 모델의 모든 매개변수를 업데이트하는 작업이 포함됩니다.
  3. 명령어 미세 조정: 명령 미세 조정에는 언어 모델이 특정 쿼리나 작업에 어떻게 응답해야 하는지 명시적으로 보여주는 예를 사용하여 언어 모델을 훈련하는 프로세스가 포함됩니다.
  4. 인간 피드백을 통한 강화 학습(RLHF): RLHF는 강화 학습 원리를 통합하여 지도 미세 조정 개념을 한 단계 더 발전시킵니다.

감독 및 비지도 미세 조정뿐만 아니라 대조 학습과 같은 기술은 LLM에만 국한되지 않으며 LLM이 출현하기 전부터 도메인 적응을 위해 사용되었습니다.

DPO - 더 간단한 접근 방식:
직접 정책 최적화(DPO)는 복잡한 보상 모델의 필요성을 피하면서 간단한 경로를 취합니다.

  1. 구현 용이성: DPO는 별도의 보상 모델을 설계하고 교육할 필요가 없어 더 많은 사용자가 접근할 수 있으므로 사용자 친화적입니다.
  2. 계산 효율성: LLM에서 직접 작동하는 DPO는 여러 단계를 포함하는 RLHF에 비해 훈련 시간이 더 빠르고 계산 비용이 낮습니다.
  3. 더 강력한 제어: 사용자는 LLM의 동작을 직접 제어하여 RLHF의 복잡성 없이 특정 목표와 선호도를 향해 LLM을 안내합니다.
  4. 빠른 수렴: DPO는 더 단순한 구조와 직접적인 최적화로 인해 원하는 결과를 더 빨리 달성하는 경우가 많으므로 빠른 반복이 필요한 작업에 적합합니다.
  5. 향상된 성능: 최근 연구에 따르면 DPO는 특히 요약 및 대화 작업에서 감정 제어 및 응답 품질과 같은 시나리오에서 RLHF보다 성능이 뛰어날 수 있는 것으로 나타났습니다.

RLHF - 더욱 구조화된 접근 방식:
강화 학습 원리를 활용하여 보다 구조화된 경로를 따릅니다.

  1. 복잡성: RLHF는 더 복잡하고 때로는 불안정할 수 있으며 더 많은 계산 리소스를 요구하고 수렴, 드리프트 또는 상관 관계가 없는 배포 문제와 같은 문제를 처리합니다.
  2. 보상 정의의 유연성: RLHF는 보다 미묘한 보상 구조를 허용하므로 LLM의 결과를 정밀하게 제어해야 하는 작업에 유용합니다.
  3. 다양한 피드백 형식 처리: RLHF는 숫자 평점이나 텍스트 수정을 포함하여 다양한 형태의 인간 피드백을 처리할 수 있는 반면, DPO는 주로 바이너리 기본 설정에 의존합니다.
  4. 대규모 데이터세트 처리: RLHF는 특히 분산 학습 기술을 사용하여 대규모 데이터세트를 처리하는 데 더 효율적일 수 있습니다.

요약하자면, 선택은 특정 작업, 사용 가능한 리소스 및 원하는 제어 수준에 따라 달라지며 두 방법 모두 서로 다른 상황에서 강점과 약점을 제공합니다.

매개변수 효율적 미세 조정(PEFT)

PEFT(매개변수 효율적인 미세 조정)는 LLM 미세 조정의 리소스 집약적 특성을 해결합니다.

PEFT(매개변수 효율적인 미세 조정)의 장점

  1. 계산 효율성: PEFT는 전체 미세 조정보다 훨씬 적은 매개변수로 LLM을 미세 조정합니다.
  2. 메모리 효율성: PEFT는 사전 학습된 모델 가중치의 대부분을 동결함으로써 모든 매개변수 수정과 관련된 과도한 메모리 사용을 방지합니다.
  3. 재앙적인 망각 완화: PEFT는 모델이 사전 훈련된 상태에서 지식을 잃는 전체 미세 조정 중에 관찰되는 현상인 치명적인 망각을 방지합니다.
  4. 양식에 따른 다양성: PEFT는 자연어 처리 작업을 넘어 컴퓨터 비전 및 오디오와 같은 다양한 양식에서 효율성이 입증되었습니다.
  5. 여러 작업을 위한 모듈식 적응: PEFT의 모듈식 특성을 통해 작은 작업별 가중치를 추가하여 동일한 사전 학습된 모델을 여러 작업에 맞게 조정할 수 있습니다.
  6. INT8 조정: PEFT의 기능에는 INT8(8비트 정수) 조정이 포함되어 있어 다양한 양자화 기술에 대한 적응성을 보여줍니다.

요약하면 PEFT는 대규모 언어 모델을 미세 조정하고 다운스트림 작업의 성능을 유지하면서 계산 및 메모리 문제를 해결하기 위한 실용적이고 효율적인 솔루션을 제공합니다.
