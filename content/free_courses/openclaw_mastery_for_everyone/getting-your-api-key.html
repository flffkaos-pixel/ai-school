API 키 받기

OpenClaw는 여러 AI 제공업체와 협력합니다.

API 키란 무엇입니까?

API 키는 OpenClaw가 귀하를 대신하여 AI 제공업체(OpenAI, Google, Anthropic)와 통신할 수 있게 해주는 고유 코드입니다.

이렇게 생각해보세요. 구독하면 귀하가 챗봇을 사용할 수 있습니다.

비용에 대한 참고 사항

OpenClaw는 상시 작동되는 에이전트입니다.

이 과정에서 토큰당 비용을 지불하는 경우 API 사용량에 대해 $20~$30를 따로 설정하는 것이 좋습니다.

참고: Anthropic Claude 구독(Pro 및 Max)에는 OpenClaw 사용이 포함되지 않습니다.

OpenClaw 커뮤니티의 많은 사람들은 API 비용을 완전히 피하기 위해 Mac mini 또는 기타 전용 하드웨어에서 로컬 모델을 실행합니다.

오픈AI(GPT)

OpenAI는 토큰당 지불 방식 대신 구독 기반 OAuth 경로도 지원하는 유일한 제공업체입니다.

API 키 받기

  1. platform.openai.com으로 이동하세요.
  2. '가입'을 클릭하고 이메일, Google, Microsoft 또는 Apple에 등록하세요.
  3. 이메일 주소를 확인하세요
  4. SMS를 통해 전화번호를 확인하세요(필수).
  5. 왼쪽 사이드바에서 'API 키'를 클릭합니다.
  6. '새 비밀 키 만들기'를 클릭하고 이름을 지정하세요.
  7. 즉시 키를 복사하세요.
  8. 설정 > 결제로 이동하여 결제 수단을 추가하고 최소 20달러의 크레딧을 로드하세요.

구글 (쌍둥이자리)

Google은 시작하기에 가장 넉넉한 무료 등급을 제공합니다.

  1. aistudio.google.com으로 이동하세요.
  2. Google 계정으로 로그인
  3. Generative AI 서비스 약관에 동의하세요.
  4. 왼쪽 사이드바에서 'API 키 가져오기'를 클릭하세요.
  5. 프로젝트를 선택하세요(또는 Google에서 기본 프로젝트를 생성하도록 하세요).
  6. '키 만들기'를 클릭하세요.
  7. 키를 복사하여 안전한 곳에 보관하세요.

인류학(클로드)

참고: Claude 구독(Pro 및 Max)에는 OpenClaw와 동일한 타사 도구 사용이 포함되어 있지 않습니다.

API 키 받기

  1. console.anthropic.com으로 이동하세요.
  2. '가입'을 클릭하고 Google, Microsoft, Apple 또는 이메일로 등록하세요.
  3. 이메일로 가입한 경우 이메일을 확인하세요.
  4. 왼쪽 사이드바에서 설정 > API 키로 이동합니다.
  5. '키 만들기'를 클릭하고 이름을 지정하세요.
  6. 즉시 키를 복사하세요.
  7. 결제 수단을 추가하고 크레딧을 로드하려면 설정 > 결제로 이동하세요.
