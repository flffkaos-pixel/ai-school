5일차 빌드: 기술 제공

이것은 5일차에 대한 사용자용 가이드입니다. 오늘은 단계별로 기술 흐름을 수행합니다.

클로허브는 OpenClaw의 공개 기술 레지스트리입니다.

운영 단계는 이 파일과 몇 개의 작은 지침 파일로 나누어져 있습니다.

시작하기 전에 필요한 것

  • 1일차 완료: OpenClaw 설치 및 보안
  • 2일차 완료: ID 파일이 생성되고 올바르게 로드됨
  • 3일차 완료: 텔레그램 연결 및 작동 중
  • 4일차 완료: 사전 예방적 워크플로가 이미 존재합니다.
  • 웹 채팅을 통해 Claw에 접근
  • 휴대폰의 텔레그램

5일차 달리기 방법

다음 순서로 파일을 살펴보세요.

  1. 채팅에서 document-summary 검사
  2. claw-instructions-install-document-summary.md
  3. claw-instructions-create-quick-note-skill.md
  4. claw-instructions-finalize-skills.md

이 순서는 공개 기술을 검사하는 방법, 안전하게 설치하는 방법, Claw에게 자신만의 행동을 가르치는 방법 등 전체 그림을 제공합니다.

1단계: ClawHub 기술 검사

다음 메시지를 복사하여 웹 채팅에 붙여넣습니다.

ClawHub에서 document-summary을 검사하고, 그것이 수행하는 작업, 어떤 종류의 요청이 있어야 하는지, 자격이 있는지 증명하거나 추가 바이너리가 필요한지 여부, 위험해 보이거나 범위를 벗어나는 모든 것을 일반적인 영어로 설명합니다.

오늘의 ClawHub 스킬 예시는 document-summary 입니다.

2단계: document-summary 설치

검사가 만족스러우면 다음 내용을 복사하여 웹 채팅에 붙여넣으세요.

https://raw.githubusercontent.com/aishwaryanr/awesome-generative-ai-guide/main/free_courses/openclaw_mastery_for_everyone/days/day-05-give-it-skills/claw-instructions-install-document-summary.md을 이해하고 모든 단계를 추가하세요.

claw-instructions-install-document-summary.md은 Claw에게 다음을 지시합니다.

  • 확인 후 이 작업 공간에 document-summary을 설치하십시오.
  • 착륙한 위치와 준비가 되었는지 확인하세요.
  • 텔레그램이나 웹 채팅에서 사용할 정확한 요청 패턴을 알려줍니다.
  • 새로운 스킬을 사용하기 전에 OpenClaw에서 /new을 입력하라고 알려줍니다.

5일차 전반부입니다. 처음부터 다시 구축하는 대신 좋은 워크플로 하나를 빌려옵니다.

육안으로 확인하려면 OpenClaw에서 Skills을 열어보세요.

이 단계 후에 계속 진행하기 전에 OpenClaw에 /new을 입력하여 새 세션을 시작하세요.

3단계: quick-note 생성

document-summary이 설치된 후 다음 내용을 복사하여 웹 채팅에 붙여넣으세요.

https://raw.githubusercontent.com/aishwaryanr/awesome-generative-ai-guide/main/free_courses/openclaw_mastery_for_everyone/days/day-05-give-it-skills/claw-instructions-create-quick-note-skill.md을 이해하고 모든 단계를 추가하세요.

claw-instructions-create-quick-note-skill.md은 Claw에게 다음과 같은 작은 맞춤 기술을 생성하라고 지시합니다.

  • note:에서 트리거됩니다.
  • 메모를 저장하기 전에 분류합니다.
  • memory/YYYY-MM-DD.md에 깨끗한 타임스탬프 항목을 저장합니다.
  • 메모에 향후 조치가 암시될 때 개방 루프 항목을 추가합니다.
  • 짧은 확인으로 답장

이것은 5일차의 후반부입니다. Claw에게 작업 방식에 맞는 한 가지 행동을 가르칩니다.

이 단계 후에 계속 진행하기 전에 OpenClaw에 /new을 입력하여 새 세션을 시작하세요.

경고]
OpenClaw 웹 인터페이스에서 추가 확인을 수행하고 Agents --> Skills으로 이동하여 해당 기술이 활성화되어 있는지 확인하세요.

4단계: 마무리 및 확인

quick-note이 생성된 후 이를 복사하여 웹 채팅에 붙여넣습니다.

https://raw.githubusercontent.com/aishwaryanr/awesome-generative-ai-guide/main/free_courses/openclaw_mastery_for_everyone/days/day-05-give-it-skills/claw-instructions-finalize-skills.md을 이해하고 모든 단계를 추가하세요.

안내파일은 다음을 알려줍니다.

  • 두 기술이 모두 존재하는지 확인
  • 5일차 스킬은 /new 이후 새로운 세션에서 사용된다는 점을 상기시켜주세요
  • 각 기술을 테스트하기 위한 정확한 메시지를 제공합니다.
  • 5일차 설정에 대해 PASS 또는 FAIL을 보고합니다.

이 시점에서 Claw에는 커뮤니티에서 재사용 가능한 워크플로 하나와 함께 만든 워크플로가 있습니다.

검증하다

웹 채팅에서 Claw에게 물어보세요:

오늘 설정한 두 가지 기술, 각각의 기술이 어디에 있는지, 각 기술을 테스트하기 위해 보내야 할 정확한 메시지, 활성화되기 전에 새로운 세션이 필요한지 여부를 알려주십시오.

문제 해결


Claw는 모든 작업을 한 번에 시작합니다.
중단하고 현재 5일차 단계를 벗어나지 말라고 지시합니다. 


검사가 모호한 느낌
'CODE0'이 무엇을 하는지, 무엇을 실행해야 하는지, 무엇에 의존하는지 알기 쉬운 영어로 설명해 달라고 요청하세요.


사용자 지정 스킬 설명이 모호한 느낌
Claw에게 텔레그램에서 보내려는 정확한 note: 트리거를 중심으로 다시 작성하도록 요청하세요.


아직 해당 스킬을 사용할 수 없는 것 같습니다.
OpenClaw에 /new을 입력하여 새 세션을 시작한 다음 다시 테스트하세요.
