5일차: document-summary 설치

이 작업 공간에 document-summary을 설치하세요.

사용자가 아직 설치를 승인하지 않은 경우 먼저 확인을 요청하세요.
이미 설치되어 준비가 된 경우 복제하는 대신 이를 보고하세요.

설치 후 사용자에게 다음과 같이 안내합니다.

  • 스킬이 살아있는 곳
  • 새로운 기술을 사용하기 전에 OpenClaw에서 /new을 입력해야 함
  • 테스트할 정확한 메시지

다음과 같은 명확한 테스트 패턴을 사용하십시오.

  • Summarize this text: <content>
  • Use document-summary on this link: <url>

설치 보고서가 완료되면 중지하세요.
