7일 차 빌드: Make It Research

이것은 7일차에 대한 사용자용 가이드입니다. 오늘은 Claw에 대한 실시간 웹 검색을 켭니다.

브라우저 도구는 Hostinger의 OpenClaw에도 내장되어 있습니다.

시작하기 전에 필요한 것

  • 1일차 완료: OpenClaw 설치 및 보안
  • 2일차 완료: ID 파일이 생성되고 올바르게 로드됨
  • 3일차 완료: 텔레그램 연결 및 작동 중
  • 4일차 완료: 사전 예방적 워크플로가 이미 존재합니다.
  • Day 5 complete: skills are working
  • 6일차 완료: 이메일 분류가 작동 중입니다.
  • Access to your Claw through the Hostinger web chat
  • A Brave Search account

7일차 달리기 방법

다음 순서대로 단계를 진행하세요.

  1. Brave Search API 키를 받으세요
  2. 채팅에서 web_search 검사
  3. claw-instructions-configure-web-search.md
  4. claw-instructions-create-research-brief.md
  5. 실력을 검증하다
  6. 하나의 실제 연구 프롬프트를 실행

이 순서는 설정을 읽기 쉽게 만듭니다.

1단계: Brave Search API 키 받기

아직 Brave Search 계정이 없다면 https://brave.com/search/api에서 시작하세요.

https://api-dashboard.search.brave.com/app/keys

검색 API 키가 필요합니다.

대시보드가 ​​낯설게 보인다면 이 페이지를 노리고 있는 것입니다.

2단계: web_search 검사

이것을 복사하여 Hostinger 웹 채팅에 붙여넣으세요.

내장이 된 web_search 도구의 기능, Brave Search가 이 도구에 어떻게 사용하는지, 어떤 유형의 연구 질문에 가장 가까운지 설명하세요.

이는 7일차의 첫 번째 유용한 변화입니다. Claw는 현재 주제에 대한 추측을 멈추고 실시간 결과를 가져오기 시작합니다.

3단계: web_search 구성

Brave API 키가 있으면 이를 복사하여 웹 채팅에 붙여넣으세요.

https://raw.githubusercontent.com/aishwaryanr/awesome-generative-ai-guide/main/free_courses/openclaw_mastery_for_everyone/days/day-07-make-it-research/claw-instructions-configure-web-search.md을 이해하고 모든 단계를 추가하세요.

claw-instructions-configure-web-search.md은 Claw에게 다음을 지시합니다.

  • 공급자 brave을 사용하여 web_search을 구성합니다.
  • Brave Search API 키를 요청하세요
  • 작업 공간 `CODE0`에 짧은 7일차 웹 조사 가드레일을 추가하세요.
  • 정확히 무엇이 변경되었는지, 어떻게 테스트하는지 알려주세요.

4단계: research-brief 생성

web_search이 작동하면 다음을 복사하여 웹 채팅에 붙여넣으세요.

https://raw.githubusercontent.com/aishwaryanr/awesome-generative-ai-guide/main/free_courses/openclaw_mastery_for_everyone/days/day-07-make-it-research/claw-instructions-create-research-brief.md을 이해하고 모든 단계를 추가하세요.

claw-instructions-create-research-brief.md은 Claw에게 다음과 같은 사용자 정의 기술을 생성하라고 지시합니다.

  • 당신이 어떤 주제에 대한 연구 개요를 요청할 때 촉발됩니다.
  • 라이브 소스에는 web_search을 사용합니다.
  • 인용문과 함께 짧고 구조화된 답변을 반환합니다.
  • 7일차 동안 검색 전용 경로 내에 유지됩니다.

이 단계 후에 OpenClaw에 /new을 입력하여 새로운 기술을 테스트하기 전에 새로운 세션을 시작하세요.
