6일차: imap-smtp-email 설치

이 작업 공간에 imap-smtp-email을 설치하세요.

사용자가 아직 설치를 승인하지 않은 경우 먼저 확인을 요청하세요.
이미 설치되어 준비가 된 경우 복제하는 대신 이를 보고하세요.

사용자에게 다음 사항을 물어보세요.

  • 개인 Gmail 주소
  • 6일차에 생성한 Gmail 앱 비밀번호

다음 제약 조건을 사용하여 6일차에 대한 기술을 구성합니다.

  • 포트 993에서 imap.gmail.com을 사용하세요.
  • 사용자의 전체 Gmail 주소를 IMAP 사용자로 사용
  • 받은편지함 읽기에 Gmail IMAP 설정 사용
  • ~/.config/imap-smtp-email/.env에 구성을 저장합니다.
  • 사용자에게 앱 비밀번호를 다시 알려주지 마세요.
  • 8일차에 전송이 처리되므로 지금은 SMTP 설정을 설정하지 않은 상태로 둡니다.
  • 기술 설정에서 아직 적용하지 않은 경우 구성 파일 권한이 소유자 전용인지 확인하세요.

설치 및 구성 후 사용자에게 다음을 안내합니다.

  • 스킬이 살아있는 곳
  • 구성 파일이 있는 곳
  • 구성 파일 권한이 올바른지 여부
  • 6일차에는 Gmail 받은편지함 읽기 전용으로 구성되었습니다.
  • 새로 추가된 기술을 시도하기 전에 OpenClaw에서 /new을 입력해야 함
  • Gmail 읽기가 작동하는지 확인하기 위해 지금 사용할 수 있는 정확한 테스트 메시지 하나

설치 보고서가 완료되면 중지하세요.
